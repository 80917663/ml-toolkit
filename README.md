# 科研数据分析工具

这是个给研究人员用的桌面软件，不用写代码就能跑机器学习模型。

## 能做什么

- 导入 Excel 或 CSV 数据
- 清洗数据（处理缺失值、异常值、标准化等）
- 快速做探索性分析（EDA）
- 训练分类或回归模型
- 查看特征重要性，用 SHAP 解释模型
- 导出图表和报告

## 运行环境

```bash
pip install -r requirements.txt
python main.py
```

推荐 Python 3.8+，内存 8GB 以上。如果数据量比较大（样本超过一万或特征上千），运行速度会慢一些。

## 使用顺序

1. **Data** - 导入数据，选择目标列
2. **Model** - 选算法，训练模型，看效果
3. **EDA** - 可选，看看数据分布和相关性
4. **Explain** - 可选，用 SHAP 解释模型
5. **Report** - 导出结果，保存在runs文件夹下

## 支持的算法

**分类**：逻辑回归、SVM、KNN、随机森林、梯度提升、朴素贝叶斯、MLP

**回归**：线性回归、岭回归、Lasso、SVR、随机森林回归、梯度提升回归、MLP回归

## 数据格式

每一行是一个样本，最后一列默认是目标变量y。支持 CSV 和 Excel 文件。

## 注意事项

- 数据量太大时 SHAP 计算会卡，建议控制在 10000 行以内
- 类别不平衡的情况目前没有特殊处理
- 报告支持导出 Markdown 格式，方便写论文

## 技术栈

Python + PySide6 + pandas + scikit-learn + shap + matplotlib
